Latitude vs Weather

This project shows which weather phoenomena correlate to proximity to the equator. It was completed using python pandas and matplotlib.

The data for the project was pulled from the OpenWeather API. I looked at latitude vs temperature, cloudiness, humidity, and wind speed. For 500 cities real-time as the project was underway in May 2021.
